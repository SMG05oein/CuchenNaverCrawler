# 쿠첸 네이버 카페 크롤링

***

## 개발 언어: 파이썬

***

#### 개발 기간: 2026년 2월 n일 ~ 2026년 3월 16일
- 1차 목표: 3월 7일까지 1차 베타테스터 내놓기
- 2차 목표: 3월 12일까지 최종본 만들고 검수 받기
- 3차 목표: 프로젝트 완료일까지 검수 받은 거 수정

---

#### 개발 규칙

- 기존에 개발 방식처럼 main branch에 push하는 건 금지.
- 시중에 나와있는 branch전략 중 선택하여 운영
- 최소한의 코드 구조화는 할 것
- 파일 구조 및 기타 등등의 변경 사항이 있을 시에 즉시 리드미 반영
